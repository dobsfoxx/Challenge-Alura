# Challenge-Alura

<p>O Challenge Alura do programa Oracle Next Education consiste em uma análise de dados de 4 lojas diferentes visando aspectos como <strong>faturamento total, média de avaliações, produtos mais e menos vendidos e média de frete,</strong> gerando um relatório com o resumo dos dados e o motivo da escolha</p>
<p>Foi utilizada a biblioteca Pandas para leitura e manipulação das bases de dados e MatPlotlib para criação de gráficos para exibição dos resultados.</p>
<p>O relatório se encontra no final do arquivo, para copiar basta fazer o download do arquivo do notebook e usar o google collab ou outra IDE que suporte arquivos com a extensão ipynb. </p>
