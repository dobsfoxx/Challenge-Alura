# Challenge-Alura

<p>O Challenge Alura do programa Oracle Next Education consiste em uma análise de dados de 4 lojas diferentes visando aspectos como <strong>faturamento total, média de avaliações, produtos mais e menos vendidos e média de frete,</strong> para decidir qual loja deverá ser fechada.</p>
<p>O relatório final se encontra no final do arquivo, para copiar basta fazer o download do arquivo do notebook e usar o google collab ou outra IDE que leia arquivos com a extensão ipynb. </p>
